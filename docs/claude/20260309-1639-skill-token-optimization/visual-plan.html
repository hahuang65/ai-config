New File

A curated extract of VE's quality-critical guidance. Contains: aesthetic directions, typography rules, color constraints, style principles, Mermaid guidance, overflow prevention, output rules, quality checks, anti-patterns, and the slop test. Commands read this instead of the 33K SKILL.md.

# Visual Explainer — Core Quality Guide ## Aesthetic Direction Constrained aesthetics (prefer): Blueprint, Editorial, Paper/ink, Monochrome terminal Forbidden: Neon dashboard, gradient mesh, Inter + violet/indigo ## Typography Forbidden as --font-body: Inter, Roboto, Arial, Helvetica, system-ui Good pairings: DM Sans/Fira Code, Instrument Serif/JetBrains Mono, ... ## Color Forbidden: #8b5cf6, #7c3aed, cyan-magenta-pink, gradient text, glowing shadows Good: Terracotta+sage, Teal+slate, Rose+cranberry, Amber+emerald, Blue+gold ## Style Principles (surfaces, depth, animation rules) ## Mermaid (theming, zoom, layout, classDef rules) ## Overflow Prevention (min-width:0, overflow-wrap, no flex on li) ## Output (docs/claude/ location, open in browser) ## AI Illustrations (surf CLI, base64, when to skip) ## Quality Checks (squint test, swap test, both themes, slop test) ## File Structure (self-contained HTML, CDN fonts only) ## Verification (fact sheet before generating)

Implementation Notes
Drift prevention. core.md has a header: "Extracted from SKILL.md — review when SKILL.md changes." The quality constraints it contains (forbidden fonts, colors, anti-patterns) change rarely. When they do change, core.md is the only file that needs updating — commands reference it, they don't inline the rules.
Do not modify SKILL.md. The full skill remains the single source of truth for /generate-web-diagram and direct VE invocations. core.md is an extract, not a replacement. Two tiers: core.md for specialized commands, SKILL.md for general-purpose.
Token budget. Before: ~681K chars per /build-feature run. After: ~318K chars. Savings come from: fact-check decoupled (~150K), 4 commands read 6K core.md instead of 33K SKILL.md (~108K), inline boilerplate compressed (~5K). References (css-patterns + libraries) are still read when genuinely needed.
