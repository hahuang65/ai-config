Layer 4

01 Orchestrator

Skill

build-feature

3-phase workflow: Research → Plan (with annotation cycles) → Implement. Produces artifacts in docs/claude/<timestamp>-<slug>/. Optional VE integration at each phase transition.
Invokes: research, plan, implement skills + /diff-review, /fact-check
disable-model-invocation
Artifacts Produced
  • research.md Phase 1 — always
  • architecture.html Phase 1 — if VE
  • plan.md Phase 2 — always
  • visual-plan.html Phase 2 — if VE
  • diff-review.html Phase 3 — if VE
invokes via Skill tool
Layer 3

02 Phase Skills

Phase 1

research

Deep-reads codebase. Writes research.md with 8 sections. Invokes /generate-architecture-diagram. STOPs for user review.
opus
Phase 2

plan

Reads research + source. Conditionally invokes architect agent and domain skills. Writes plan, runs annotation cycles, generates todo list. Invokes /generate-visual-plan.
opus
Phase 3

implement

Reads plan. Runs 13-step process: TDD batches → verify loop → DB review → simplify → refactor → code review → doc update → fact-check → visual refresh.
sonnet
invoke via Agent tool + Skill tool

Dead code detection. SAFE/CAREFUL/RISKY categories.
Rules: coding-style
sonnet
Database

database-reviewer

Conditional: DB code touched. EXPLAIN ANALYZE, N+1, RLS, indexes.
Rules: security performance
sonnet
Docs

doc-updater

Conditional: significant changes. Syncs READMEs, CHANGELOG, architecture docs.

sonnet
Layer 2b

04 Commands

/generate-architecture-diagram
Blueprint/editorial HTML. Reads core + css-core + css-mermaid + libraries + architecture template.
/generate-visual-plan
State machines, code snippets, edge cases. Reads core + all 3 CSS modules + libraries.
/diff-review
Before/after panels, KPI, decision log. Reads core + all 3 CSS modules + libraries.
/fact-check
Extracts claims, verifies against code/git. Corrects in place. No VE refs needed.
/simplify
Built-in Claude Code skill. No local command file. Reviews reuse, quality, efficiency.
/plan-review
Risk assessment. Reads core + all 3 CSS modules + libraries.
/project-recap
Mental model snapshot. Reads core + all 3 CSS modules + libraries.
/generate-web-diagram
General-purpose HTML diagram. Reads core + all 3 CSS modules + libraries.
read reference files
