B1–B2 Remove Redundant Rule Reads

Rules are already loaded as CLAUDE.md global instructions. Remove the "read all rule files" instruction from plan/SKILL.md and implement/SKILL.md. Saves ~12K in redundant disk reads.

B3 Batch tdd-guide Calls

Invoke tdd-guide with 3-5 related tasks per invocation instead of one. For 20 todos: 20 invocations drop to ~5. Each saves ~4.6K in overhead. Total: ~69K saved.

B4 Merge Security into Code Review

Both agents read the same files. Code-reviewer already has a Security (CRITICAL) checklist. Merge adds OWASP Top 10 checks + auto-fix policy. Delete security-reviewer.md. Saves ~15K.

B5 Conditional Architect

Skip architect for simple features (bug fixes, single-file changes, UI tweaks). Run only for multi-module features, API design, or DB schema changes. Saves ~15K for ~30-40% of features.
