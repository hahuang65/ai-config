Medium Impact

Specialize /research diagram

~27.5K saved per /research
Agent isolation is the unsung hero. The implement phase invokes 5-6 agents (tdd-guide, code-reviewer, security-reviewer, refactor-cleaner, database-reviewer, doc-updater) totaling ~18K of instruction text. Because each runs in its own context window, the main conversation pays zero tokens for agent instructions. Only agent results flow back.
